# Non-Commercial Intent

This project is licensed under [GPL v3](LICENSE) — the standard free-software
copyleft license — which legally permits commercial use, modification, and
redistribution as long as derivative works keep the same license.

**But that is NOT the spirit of this project.**

The author's clearly stated intent is:

> **This is a free toolkit. It must remain free.**
>
> Do NOT use this software, in whole or in part, to:
>
> 1. Sell pre-built ROM images that bundle copyrighted game data.
> 2. Sell flashed Yamanooto cartridges loaded with games (whether using this
>    toolkit, scripts derived from it, or images produced by it).
> 3. Charge any user for "the service" of building or flashing a ROM with this
>    toolkit, beyond reasonable reimbursement of hardware costs.
>
> If you see anyone profiting from a service or product that uses this
> toolkit, **they are scamming you** — the tool itself is and will always be
> free.

The launcher displays this notice on every boot (splash + scrolling marquee)
specifically because some people will inevitably try to monetise this work.
Calling them out is part of the toolkit.

If you fork this project, **please keep the notice prominent**. It is the
single most important non-technical contribution this repository makes.

## Why GPL v3 then?

Because legally restricting "commercial use" creates an OSI-incompatible
license that fragments the ecosystem. GPL v3 + a public moral position is a
deliberate choice: it lets honest people build on the code while making the
scam attempts ethically (and reputationally) untenable.

— Antxiko
